# **App Name**: Caddie Connect

## Core Features:

- Availability Submission: Caddies can submit their daily availability through a simple form.
- Assigned Schedule View: Caddies can view their personal work schedule.
- Broadcast Announcements: Receive announcements from the caddiemaster.

## Style Guidelines:

- Primary color: Royal blue (#4169E1) to give a professional and trustworthy impression.
- Background color: Light blue (#E6F0FF), a very desaturated tint of the primary hue to provide a clean backdrop.
- Accent color: Firebrick red (#B22222) to highlight important notifications and calls to action.
- Body and headline font: 'PT Sans', a sans-serif for clear readability.
- Use golf-related icons.
- Clean and simple layout.
- Subtle animations, such as fading or sliding transitions, can be used to enhance the user experience when navigating between different views or when new information is loaded.
